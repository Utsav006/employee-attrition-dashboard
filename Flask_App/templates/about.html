System Architecture & Algorithm Comparison

During the development lifecycle, the dataset was mathematically balanced using SMOTE (Synthetic Minority Over-sampling Technique). Following this, three distinct machine learning architectures were trained and evaluated to ensure the most mathematically sound deployment.

1. Random Forest Classifier

Random Forest is an ensemble learning method that constructs a multitude of decision trees during training. It outputs the class that is the mode of the classes selected by individual trees. While excellent for preventing overfitting, it achieved the lowest F1 Score (0.5821) on our specific HR dataset, struggling slightly to separate the complex boundaries of employee attrition.

2. XGBoost Classifier

Extreme Gradient Boosting (XGBoost) is a highly sophisticated, sequential ensemble algorithm. It builds decision trees one by one, where each new tree attempts to correct the errors made by the previous tree. It performed very well, achieving an F1 Score of 0.6105, but fell slightly short of the simpler linear architecture.

3. Logistic Regression (Primary Deployed Engine)

Logistic Regression emerged as the superior algorithm for this environment, achieving the highest validation F1 Score (0.6374) and ROC-AUC (0.7712). Rather than building trees, it calculates a linear combination of all employee features based on learned weights.

z = (β₀) + (β₁ × Age) + (β₂ × Income) + ... + (βₙ × Featureₙ)

This result is then passed through the Sigmoid Activation Function to compress the raw number into a valid probability format between 0% and 100%:

P(Attrition = Yes) = 1 / (1 + e⁻ᶻ)

Because employee behavior in this dataset possesses clear linear relationships (e.g., lower income consistently correlates to higher attrition risk), this transparent mathematical model generalized better to unseen data than the complex tree-based alternatives.
